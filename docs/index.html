The part nobody else does

Reasoning, benchmarked — not just tokens

Every tool claims it "preserves meaning." We test it.

Compression that shrinks tokens but quietly breaks the model's reasoning is worse than useless. So we ship a paired GSM8K answer-stability benchmark: the same math problems run through Claude with and without reduction, at every level, reporting whether the answers change — with Wilson confidence intervals, an exact McNemar test, and honest sample-size math. It even documents its own limits. That's the credibility: numbers you can re-run, not a slogan.
